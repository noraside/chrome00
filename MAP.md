# MAP

  Main folders for the Gramado OS

## Bootloader

```
boot/arch/x86/bsp/
```

## Kernel

```
kernel/
modules/hvmod0/
init/ (The init process)
```

## Posix-like commands

```
cmds/
```

## GUI applications

```
winu/core/ds/ds00/ (The display server)
apps/              (Client-side GUI applications)
```
